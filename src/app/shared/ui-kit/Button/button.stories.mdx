## "As" Prop

You can specify what element to use.

```jsx
// Using tag name
<Button as="a" href="#">Link</Button>

// Using another component (should pass className prop in order styles to be applied)
<Button as={Link} to="#">Router Link</Button>
```

## Props

Button also accepts all dom props depending on html tag (button, a, etc) it uses.
